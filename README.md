# CatLab

Aplicação web para geração de imagens de gatinhos, desenvolvida com foco em estudo e demonstração de habilidades em desenvolvimento front-end.

---

## 📌 Sobre o projeto

O **CatLab** é uma aplicação voltada à geração de imagens de gatos, criada com o objetivo de aplicar conceitos modernos de desenvolvimento front-end, organização de código e boas práticas de programação e engenharia de software.

Este projeto também serve como base para experimentação com diferentes frameworks, permitindo a comparação e evolução entre abordagens utilizando **React** e **Angular**.

---

## 📂 Estrutura do projeto

```bash
/
├── _docs/     # Documentação e análise do projeto (.pdf)
├── angular/   # Aplicação front-end em Angular
├── react/     # Aplicação front-end em React
└── README.md
```
